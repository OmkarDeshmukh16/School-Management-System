Smart School Management System — Full-Stack SaaS Platform (MERN)

A production-grade, multi-role school management platform built with the MERN stack — designed to digitize and automate the end-to-end administrative workflow of an educational institution.

The system handles everything from student enrollment to document generation, fees, attendance, and grievances — replacing manual paperwork with a single unified dashboard.

Key features shipped:
· Role-based access control with 3 separate login portals — Admin, Student, and Clerk — each with scoped permissions and dashboards
· Automated document generation — Leaving Certificates and Bonafide Certificates produced instantly from stored student data, eliminating manual drafting
· Bulk data upload — import up to 1,000 student records in a single operation via structured file upload
· Financial Ledger — fee payment tracking with real-time individual payment status per student
· Academic Suite — marks entry, report generation, and attendance management for faculty
· Grievance Management — students can raise complaints; admins can track and resolve them through the portal
· Notice Board — broadcast official notices and holiday alerts to all users instantly

Built as a SaaS product with tiered pricing (Foundation · Professional · Enterprise), serving institutions up to 1,000+ scholars.

Stack: React.js · Node.js · Express.js · MongoDB · JWT Auth · REST API · Role-Based Access Control
